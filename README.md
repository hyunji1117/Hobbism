## 🛠️ 로컬 개발 환경 설정

### **🔗 [Hobbism 바로가기](https://hobbism-marketplace.vercel.app/)** - 취미 라이프스타일 플랫폼을 경험해보세요!

> Google/Kakao/Naver 소셜 로그인으로 별도 설치 없이 바로 이용 가능합니다.

<br />

### 사전 요구사항

- Node.js 18.0.0 이상 (테스트 완료: v20.16.0)
- npm 또는 yarn
- **환경변수 설정 필수** (소셜 로그인 연동을 위해 필요)

<br />

## 👩‍🔧 설치 및 실행 안내

### 1. 프로젝트 클론

```bash
git clone https://github.com/hyunji1117/Hobbism.git
```

### 2. 프로젝트 폴더로 이동

```bash
cd Hobbism
```

### 3. 패키지 설치

```bash
npm install
```

### 4. 환경변수 설정

```bash
touch .env.local
```

#### ⚠️ `.env.local` 파일을 생성하여 다음 값들을 설정 해주세요.

#### 소셜 로그인 (필수)

```bash
NEXT_PUBLIC_GOOGLE_CLIENT_ID=your_google_client_id
NEXT_PUBLIC_KAKAO_CLIENT_ID=your_kakao_client_id
NEXT_PUBLIC_NAVER_CLIENT_ID=your_naver_client_id
```

#### API 서버

```bash
NEXT_PUBLIC_API_BASE_URL=your_api_server_url
```

### 5. 개발 서버 실행

```bash
npm run dev
```

<br />

#### 🔑 환경변수 발급 방법

- **Google**: [Google Cloud Console](https://console.cloud.google.com/)
- **Kakao**: [Kakao Developers](https://developers.kakao.com/)
- **Naver**: [네이버 개발자센터](https://developers.naver.com/)

> **⚠️ 중요**: 환경변수 없이는 로그인이 불가능하며, 로그인 없이는 플랫폼 이용이 제한됩니다.
